# 安装

使用`git clone`命令安装:
```
git clone git@github.com:helloyongwei/AJAX-demo.git
```
# 运行

进入`AJAX-Promise-demo`目录, 并执行:
```
node server.js 8888
```
`8888`是端口, 你可以自行指定端口

在浏览其输入如下URL即可:
```
http://localhost:8888
```

你会看到一个页面: 
![响应前的图片](/imgs/response.png)

打开控制台, 点击页面的按钮. 你会看到控制台(图中红框部分)输出如下内容:
